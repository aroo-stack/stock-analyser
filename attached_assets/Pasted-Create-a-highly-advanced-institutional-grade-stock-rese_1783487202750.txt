The app must deliver highly accurate data and handle dynamic low, base, and high-case projections natively without hardcoding. 

Implement the following five specific modules and structural logic:

1. DYNAMIC SCENARIO ENGINE & INTERACTIVE UI SLIDER

Moving the slider between the Low, Base, and High targets must dynamically recalculate and display the implied Forward P/E multiple and the required revenue growth rate needed to hit that specific price target based on the current EPS ($Price = Projected\ EPS \times Multiple$).

2. AUTOMATED DATA NORMALISATION AND FILTERING PIPELINE
- Write a backend Python utility function to sanitize noisy analyst target data. It must evaluate incoming targets and filter out stale values (older than 90 days) or statistical anomalies that deviate massively from the current spot price.
- Use this template structure for the data cleanup logic:
  def filter_and_calculate_targets(analyst_data, current_price):
      valid_targets = [t for t in analyst_data if t.get('age_days', 0) <= 90]
      if not valid_targets: return {"high": current_price, "low": current_price, "high_return": 0, "low_return": 0}
      high_case = max([t['target'] for t in valid_targets])
      low_case = min([t['target'] for t in valid_targets])
      return {
          "high": high_case, "low": low_case,
          "high_return": ((high_case - current_price) / current_price) * 100,
          "low_return": ((low_case - current_price) / current_price) * 100
      }
- Display these calculated "Implied Return %" metrics clearly next to the price targets on the frontend UI.

Avoid generic commentary and focus on specific data points like capacity, margins, or CapEx."

- The table must pull the consensus 1-year and 2-year forward Earnings Per Share (EPS) estimates.
- Structure 3 static rows for Low, Base, and High cases. Include interactive input fields within the table cells allowing users to manually adjust the assumed P/E multiples (e.g., 10x, 15x, 25x) to instantly update the implied future stock price directly in the table matrix.
