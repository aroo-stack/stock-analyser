If the cache is stale or missing, fetch fresh data as normal and store the new result in the cache
Add a cached: boolean and cachedAt: string field to the analysis response so the frontend can show when data was last fetched

5. Stock Q&A chat (ask the AI about this stock)

Add a new route file routes/chat.ts

A system prompt: "You are a helpful stock analysis assistant. The user is asking about [ticker]. Here is the current analysis context: [analysisContext]. Answer in plain English, be concise, and always remind the user you are not a financial advisor."
The full conversation history from the messages table as the messages array

Register the new router in routes/index.ts
Add the two new endpoints to lib/api-spec/openapi.yaml
Frontend (artifacts/stock-analyser/src):
