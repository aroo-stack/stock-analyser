### FORMULA IMPLEMENTATIONS — implement every single one of these in pure TypeScript

#### Helper: EMA
```
multiplier = 2 / (period + 1)
ema[0] = prices[0]
ema[i] = (prices[i] - ema[i-1]) × multiplier + ema[i-1]
Return array of EMA values
```

#### Helper: SMA
```
For each position i >= period-1:
  sma[i] = average of prices[i-period+1 .. i]
```

#### Helper: Standard Deviation (population)
```
mean = sum(values) / n
variance = sum((v - mean)^2) / n
stdDev = sqrt(variance)
```

#### RSI (14-period Wilder)
```
Compute daily changes: change[i] = close[i] - close[i-1]
gains[i] = max(change[i], 0)
losses[i] = max(-change[i], 0)
Seed: avgGain = mean(gains[1..14]), avgLoss = mean(losses[1..14])
For i > 14:
  avgGain = (avgGain × 13 + gains[i]) / 14
  avgLoss = (avgLoss × 13 + losses[i]) / 14
RS = avgGain / avgLoss  (guard divide-by-zero: if avgLoss = 0, RSI = 100)
RSI = 100 - (100 / (1 + RS))
Return final RSI value. Need at least 15 data points.
```

#### Stochastic Oscillator (14-period, 3-period %D)
```
For each i >= 13:
  high14 = max of close[i-13..i]   (use close as proxy for high if no OHLC)
  low14  = min of close[i-13..i]
  %K[i]  = 100 × (close[i] - low14) / (high14 - low14)
  Guard: if high14 = low14, %K = 50
%D = 3-period SMA of %K
Return final %K and %D values.
```

#### Williams %R (14-period)
```
high14 = max(close[last 14])
low14  = min(close[last 14])
%R = (high14 - currentClose) / (high14 - low14) × -100
Guard: if high14 = low14, return -50
Range -100 to 0. Below -80 = oversold, above -20 = overbought.
```

#### MACD (12, 26, 9)
```
ema12 = EMA(closes, 12)
ema26 = EMA(closes, 26)
macdLine[i] = ema12[i] - ema26[i]  (only where both exist)
signalLine = EMA(macdLine, 9)
histogram = macdLine - signalLine (final values)
Classify:
  "bullish"  if macdLine > signalLine AND histogram > 0
  "bearish"  if macdLine < signalLine AND histogram < 0
  "neutral"  otherwise
Need at least 35 data points.
```

#### Bollinger Bands (20-period, 2σ)
```
middle = SMA(closes, 20)   — use last 20 closes
stdDev = population stdDev of last 20 closes
upper  = middle + 2 × stdDev
lower  = middle - 2 × stdDev
position = (currentPrice - lower) / (upper - lower) × 100
Guard: if upper = lower, position = 50
```

#### ATR (14-period Wilder, close-only approximation)
```
tr[i] = abs(close[i] - close[i-1])
Wilder smoothing (same as RSI):
  atr[13] = mean(tr[0..13])
  atr[i]  = (atr[i-1] × 13 + tr[i]) / 14
Return final ATR value.
```

#### OBV
```
obv = 0
For i from 1 to n-1:
  if close[i] > close[i-1]: obv += volume[i]
  if close[i] < close[i-1]: obv -= volume[i]
  (equal: no change)
obvTrend: compare average OBV last 5 days vs average OBV 10–20 days ago
  rising if recent avg > older avg by > 1%
  falling if recent avg < older avg by > 1%
  flat otherwise
```

#### ROC (20-period)
```
roc20 = (close[last] - close[last-20]) / close[last-20] × 100
Need at least 21 data points.
```

#### Moving Averages vs Price
```
ma20  = SMA(last 20 closes)
ma50  = SMA(last 50 closes)
ma200 = SMA(last 200 closes)
vsMA20  = (currentPrice - ma20)  / ma20  × 100
vsMA50  = (currentPrice - ma50)  / ma50  × 100
vsMA200 = (currentPrice - ma200) / ma200 × 100
goldenCross = ma50 > ma200   (true = bullish, false = death cross)
```

#### Fibonacci Retracement
```
high = fiftyTwoWeekHigh
low  = fiftyTwoWeekLow
levels = {
  "0%":    high,
  "23.6%": high - 0.236 × (high - low),
  "38.2%": high - 0.382 × (high - low),
  "50%":   high - 0.500 × (high - low),
  "61.8%": high - 0.618 × (high - low),
  "78.6%": high - 0.786 × (high - low),
  "100%":  low,
}
Find which two adjacent levels the current price falls between.
Return e.g. "Between 38.2% and 50% retracement"
If price > high: "Above 52-week high"
If price < low:  "Below 52-week low"
```

#### Ichimoku Cloud
```
tenkanSen  = (max(close, 9)  + min(close, 9))  / 2
kijunSen   = (max(close, 26) + min(close, 26)) / 2
senkouA    = (tenkanSen + kijunSen) / 2
senkouB    = (max(close, 52) + min(close, 52)) / 2
cloudTop    = max(senkouA, senkouB)
cloudBottom = min(senkouA, senkouB)
Signal:
  "above_cloud" if currentPrice > cloudTop
  "below_cloud" if currentPrice < cloudBottom
  "in_cloud"    otherwise
Need at least 52 data points.
```

#### Graham Number
```
Only compute if eps > 0 AND bookValuePerShare > 0:
  grahamNumber = sqrt(22.5 × eps × bookValuePerShare)
  grahamVsCurrent = (grahamNumber - currentPrice) / currentPrice × 100
    (negative = stock is above Graham Number = expensive by this metric)
    (positive = stock is below Graham Number = potentially undervalued)
```

#### Earnings Yield
```
earningsYield = (1 / peRatio) × 100
```

#### Historical Volatility (30-day, annualised)
```
logReturns[i] = ln(close[i] / close[i-1])
Use last 30 logReturns
vol = stdDev(logReturns) × sqrt(252) × 100
```

#### Sharpe Ratio (annualised, risk-free = 4.5%)
```
dailyRF = 0.045 / 252
logReturns = ln(close[i]/close[i-1]) for last 252 days
excessReturns[i] = logReturn[i] - dailyRF
sharpe = mean(excessReturns) / stdDev(excessReturns) × sqrt(252)
Guard: if stdDev = 0, return null
```

#### Sortino Ratio
```
Same excessReturns as Sharpe
downsideReturns = excessReturns where value < 0
downsideDeviation = stdDev(downsideReturns) × sqrt(252)
sortino = mean(excessReturns) × sqrt(252) / downsideDeviation
Guard: if downsideDeviation = 0, return null
```

#### Maximum Drawdown
```
peak = close[0]
maxDD = 0
For each close:
  if close > peak: peak = close
  dd = (peak - close) / peak
  if dd > maxDD: maxDD = dd
Return maxDD × 100  (as positive percentage)
```

#### Value at Risk (Historical Simulation, 95%)
```
logReturns = last 252 daily log returns, sorted ascending
VaR95 = abs(logReturns[floor(0.05 × n)]) × 100
This is the daily loss exceeded only 5% of days historically.
```

#### Calmar Ratio
```
annualisedReturn = (close[last] / close[0])^(252 / n) - 1
calmar = annualisedReturn / (maxDrawdown / 100)
Guard: maxDrawdown = 0 → return null
```

#### Skewness (sample)
```
n = length, mean = mean(returns), s = stdDev(returns)
skewness = (n / ((n-1)(n-2))) × Σ((r - mean)/s)^3
```

#### Excess Kurtosis (sample)
```
kurtosis = [n(n+1)/((n-1)(n-2)(n-3))] × Σ((r-mean)/s)^4
           - 3(n-1)^2/((n-2)(n-3))
Normal distribution = 0. Positive = fat tails.
```

### PRICE TARGET METHODS — implement all four, weighted average

#### Method 1: Monte Carlo Geometric Brownian Motion (weight 30%)
```
Inputs: logReturns from last 252 days
drift = mean(logReturns)
vol   = stdDev(logReturns)

Box-Muller normal random:
  u1 = Math.random(), u2 = Math.random()
  z  = sqrt(-2 × ln(u1)) × cos(2π × u2)

Run 10,000 paths:
  For each path, simulate N trading days:
    price[0] = currentPrice
    price[t] = price[t-1] × exp((drift - 0.5 × vol²) + vol × z)

N = 21  for 1 month
N = 126 for 6 months
N = 252 for 1 year

From 10,000 final prices, sort and take:
  median (50th pct) = main target
  10th pct          = bear case (low)
  90th pct          = bull case (high)
```

#### Method 2: Analyst Consensus Interpolation (weight 25%)
```
If analystTargetPrice available:
  oneYear   = analystTargetPrice
  sixMonths = currentPrice + (analystTargetPrice - currentPrice) × 0.65
  oneMonth  = currentPrice + (analystTargetPrice - currentPrice) × 0.15
Otherwise: skip and redistribute weight to other three equally.
```

#### Method 3: Mean Reversion to Historical P/E (weight 25%)
```
targetPE = 18   (long-run S&P 500 average — Shiller CAPE long-run mean)
If peRatio and eps both available:
  fairValue = eps × targetPE
  gap = fairValue - currentPrice
  Ornstein-Uhlenbeck approximation:
    oneYear   = currentPrice + gap × 0.30   (30% reversion in 1 year)
    sixMonths = currentPrice + gap × 0.17
    oneMonth  = currentPrice + gap × 0.03
Otherwise: skip.
```

#### Method 4: Jegadeesh-Titman Momentum Extrapolation (weight 20%)
```
Momentum persistence finding: 3–12 month momentum predicts next 3–12 months,
then reverses (Jegadeesh & Titman 1993).

threeMonthReturn = (close[last] - close[last-63])  / close[last-63]
sixMonthReturn   = (close[last] - close[last-126]) / close[last-126]

Decay factors (momentum decays and reverses after 12 months):
  oneMonth   = currentPrice × (1 + threeMonthReturn / 3)
  sixMonths  = currentPrice × (1 + sixMonthReturn × 0.70)
  oneYear    = currentPrice × (1 + sixMonthReturn × 0.35)

Need at least 126 data points for 6m return; fall back to 3m only if fewer.
```

#### Combine methods
```
For each horizon, take weighted average of available methods.
If a method is unavailable, redistribute its weight proportionally to the others.
Round all targets to 2 decimal places.
Return null for a horizon only if NO methods could produce a value.

Also return methodsUsed: string[] listing which methods contributed.
```

### One-Liner and Target Rationale — generated from computed values (not AI)

```typescript
function buildOneLiner(r: BottomLineResult): string {
  const m = r.scores.momentum, v = r.scores.valuation,
        ri = r.scores.risk,    q = r.scores.quality;

  const mStr  = m  > 65 ? "strong momentum"       : m  < 40 ? "weak momentum"    : "neutral momentum";
  const vStr  = v  > 65 ? "attractive valuation"  : v  < 40 ? "stretched valuation" : "fair valuation";
  const rStr  = ri > 65 ? "low risk profile"      : ri < 40 ? "elevated risk"    : "moderate risk";
  const qStr  = q  > 65 ? "solid fundamentals"    : q  < 40 ? "weak fundamentals": "average fundamentals";

  const rsiNote = r.signals.rsi14 != null
    ? r.signals.rsi14 > 70 ? " RSI is overbought."
    : r.signals.rsi14 < 30 ? " RSI is oversold — could be a buying opportunity."
    : "" : "";

  const volNote = r.signals.historicalVolatility30d != null && r.signals.historicalVolatility30d > 50
    ? " High volatility means large price swings are likely." : "";

  return `This stock shows ${mStr}, ${vStr}, ${rStr}, and ${qStr}.${rsiNote}${volNote}`;
}

function buildTargetRationale(r: BottomLineResult, currentPrice: number): string {
  const used = r.methodsUsed.join(", ");
  const oneY = r.priceTargets.oneYear;
  if (!oneY) return "Insufficient data to generate reliable price targets.";
  const pct = ((oneY - currentPrice) / currentPrice * 100).toFixed(1);
  const dir = oneY > currentPrice ? "upside" : "downside";
  return `The 1-year target of $${oneY.toFixed(2)} implies ${Math.abs(Number(pct))}% ${dir} from the current price, derived from ${used}.`;
}
```
